= couch_crumbs

* http://github.com/lmlo/couch_crumbs

== DESCRIPTION:

An early/small library for CouchDB-based apps.

== FEATURES/PROBLEMS:

- Named properties
- Automatic timestamps (created_at/updated_at)
- Per-class database settings
